# Paleta de Cores – GRF Incorporadora

Este documento consolida a paleta oficial de cores da **GRF Incorporadora**, visando facilitar consultas futuras, padronização visual e uso consistente em produtos digitais, materiais institucionais e sistemas.

---

## Cor de Fundo

### Branco

- **HEX:** `#FFFFFF`
- **RGB:** `rgb(255, 255, 255)`
- **Uso recomendado:** fundo principal, áreas de respiro, documentos, interfaces claras.

---

## Cores Institucionais

### Azul Escuro (Principal)

- **HEX:** `#324598`
- **RGB:** `rgb(50, 69, 152)`
- **Uso recomendado:** logotipo, títulos, elementos estruturais, destaques institucionais.

### Azul Médio

- **HEX:** `#465EBE`
- **RGB:** `rgb(70, 94, 190)`
- **Uso recomendado:** botões, ícones, estados ativos, elementos de apoio ao azul principal.

### Turquesa / Verde‑Azulado

- **HEX:** `#19B4BE`
- **RGB:** `rgb(25, 180, 190)`
- **Uso recomendado:** detalhes visuais, gráficos, indicadores, elementos de inovação.

### Azul Claro / Ciano

- **HEX:** `#6ED8E0`
- **RGB:** `rgb(110, 216, 224)`
- **Uso recomendado:** fundos secundários, realces suaves, estados informativos.

---

## Observações de Uso

- A paleta mantém forte identidade tecnológica e institucional, mesmo com fundo claro.
- Recomenda‑se priorizar o **Azul Escuro** como cor dominante da marca.
- As cores claras devem ser usadas com moderação para preservar contraste e legibilidade.
- Para aplicações digitais, validar contraste conforme **WCAG 2.1** quando houver texto sobre cores coloridas.
