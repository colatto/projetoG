# Supabase

Diretorio reservado para os artefatos de plataforma ligados ao Supabase.

## Conteudo esperado

- configuracoes de ambiente e convencoes de projeto;
- migracoes de banco;
- seeds e utilitarios operacionais;
- politicas, funcoes auxiliares e suporte de autenticacao quando necessario.

## Regra local

Supabase apoia persistencia e identidade, mas nao substitui a API dedicada para regra critica de negocio e orquestracao.
