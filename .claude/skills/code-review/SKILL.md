# Skill: Code Review

## Objetivo

Revisar alteracoes com foco em:

- regressao funcional;
- quebra de regras do PRD;
- falhas de RBAC;
- riscos de integracao e auditoria;
- ausencia de testes em fluxos criticos.

## Checklist minimo

- regra critica permaneceu no backend ou dominio;
- autorizacao foi validada por perfil;
- mudancas preservam rastreabilidade;
- falhas de integracao sao tratadas;
- contratos com Sienge nao foram alterados sem validacao.
